javascript用栈的思想实现数制转换

发表于 2015-11-26   |   分类于 front-end   |  

可以利用栈将一个数字从一种数制转换成另一种数制。假设想将数字n转换为以b为基数的数字，实现转换的算法如下：

  1. 最高位为 n%b ，将此位压入栈。
  2. 使用 n/b 代替n。
  3. 重复步骤1和2，直到n等于0，且没有余数。
  4. 持续将栈内元素弹出，直到栈为空，依次将这些元素排列，就得到转换后的数字的字符串形式。
